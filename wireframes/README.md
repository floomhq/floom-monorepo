# Wireframes (reference)

HTML or static wireframes stored here are **design references**, not the live product.

Some older build-flow wireframes showed **Scheduler** and **Webhook** template cards as if they were first-class add-app ramps. Those triggers are **P3** on the roadmap (`docs/ROADMAP.md`); they are **not** exposed as template cards on the current **`/build`** page, which only ships the **GitHub** and **OpenAPI** ingest ramps (with **Docker** import still marked *Coming soon* in `BuildPage.tsx`).
