## Useful session-level notes for the next agent

- **Run tests with Node 20** locally (`volta run --node 20 …` or nvm) — `better-sqlite3` was compiled for NODE_MODULE_VERSION 115; your system Node may be 141 and will fail to load. CI uses Node 20.
- **Stress tests run via `tsx`**, not raw node — e.g. `pnpm exec tsx test/stress/test-file-inputs.mjs`, not `node --experimental-strip-types …`.
- **Don't trust `wireframes.floom.dev/v17/` for font/color**; the in-repo `apps/web/src/styles/wireframe.css` is the canonical decision log.
- **When editing `HeroDemo.tsx`**, remember the canvas is a fixed 580px tall and all three states (`EditorSurface`, `DeploySurface`, `RunSurface`) stack via `position: absolute` in `CANVAS_STYLE`. Don't add height-dependent layouts.
